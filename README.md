React를 사용하여 만든 To-do list입니다.

## To-do list

할 일을 리스트에 올리고, 할 일이 끝나면 체크할 수 있으며, 삭제할 수 있습니다. CRUD 중 Update를 제외한 나머지가 작동합니다.

## 시작하기

`npm start`를 terminal에서 실행합니다. 실행 전, npm과 NodeJS가 설치되어야 합니다.
